描述：在 Admin-FE 的门店面板内提供门店级别的订单列表视图，支持按日期范围、订单状态筛选与快捷状态按钮。

验收标准：
- 门店面板点击“订单”后能列出该门店在选定时间范围内的订单；
- 支持按订单状态（待付款/待发货/已完成/已取消等）筛选；
- 快捷状态按钮（例如“仅已付款”）生效并与分页结合使用；
- 随机抽样 10 条数据与后端 API 返回一致。

建议负责人：前端（Admin-FE）+ 后端（order list API）
预计工作量：3-5 人日

实现备注：后端需提供按 `store_id` + 时间范围 + 状态 的分页接口；前端展示按门店过滤的分页表格并接入现有的加载/禁用态工具。
